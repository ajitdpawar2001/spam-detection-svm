# 📩 Spam Detection using SVM

## 🚀 Overview
This project is a machine learning-based spam detection system using Support Vector Machine (SVM).

## 🧠 Features
- TF-IDF Vectorization
- SVM Classification
- Streamlit UI
- Real-time prediction

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
